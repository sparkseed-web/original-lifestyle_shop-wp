## オリジナルサイト(架空の雑貨ショップ)
（WordPress テーマ）

このリポジトリは、やさしく自然な雰囲気を大切にして作った  
オリジナルサイトを WordPress テーマとして再構築したものです。

静的サイトとして作ったデザインを、  
「管理画面から編集できるサイト」に育てていくことを目指しました。

---

## 🌱 使用しているもの
- WordPress
- PHP
- Advanced Custom Fields（ACF）
- HTML / CSS

必要なところだけ PHP を使いながら、  
WordPress の仕組みに合わせてテーマを組み立てています。

---

## 🌼 このテーマでできること

### ■ 商品を管理するための投稿タイプ
- `product`  
やさしい雑貨やインテリアを登録するための投稿タイプです。

### ■ 商品カテゴリの分類
- `product_category`  
カテゴリーごとに商品を整理できます。

### ■ ACF で入力できる商品情報
商品ごとに、次のような仕様を入力できます。

- サイズ  
- 座面高さ  
- 素材  
- カラー  
- 質量  
- 耐荷重  
- 組み立て  
- 生産国  
- 耐熱温度  
- 注意事項  
- 容量  
- ページ数  
- 表紙  
- 冊数  
- 用紙  

雑貨や本、食器など、  
さまざまなアイテムに対応できるようにしています。

---

## 🌸 このテーマを作った目的
- 静的サイトを WordPress 化する流れを学ぶ  
- カスタム投稿タイプや ACF を使って、編集しやすいサイトを作る  
- テンプレート階層や PHP の基本を理解する  
- 静的サイトをそのまま WordPress に落とし込む  

---

## 💡 補足
このリポジトリには WordPress 本体や `wp-config.php` は含めていません。  
テーマフォルダ（`sotto-site`）のみを管理しています。
